# Potrero — Compañera de Campo

> Aplicación offline para la salida de campo a El Potrero (Ecología General, UNLP).

**Documentación completa:** → [PROJECT.md](./PROJECT.md)

## Resumen

- **Qué es:** App web 100% offline para la salida de campo de 3 días
- **Stack:** React 19 + Vite + GitHub Pages
- **Idioma:** Solo español
- **Fuente de datos:** `/sdcard/SalidaPotrero/` (PDFs, DOCX, DOC, PPTX)
- **Relación con setae.js:** Linkea a la clave de hormigas de setae.js cuando hay internet
